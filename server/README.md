# Question 1 server

The Node/Express.js API for question 1.

## Stack used
- TypeScript
- Node.js
- Express.js

## Run locally

The following steps will allow you to run the server locally

- Install dependencies

```sh
npm install
```

- Build for `tsc` build files

```sh
npm run build
```

- Lastly, run the server

```sh
npm run dev
```
